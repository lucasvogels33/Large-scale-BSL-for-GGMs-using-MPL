This folder contains the code to reproduce the tables and figures of the article "Large-scale Bayesian Structure Learning for Gaussian Graphical Models using Marginal Pseudo-likelihood". 

It contains three folders:
1. Human gene data set (p=100): this folder contains the code to reproduce the tables and figures of Subsection 6.1 
2. Mice gene data set (p=623): this folder contains the code to reproduce the tables and figures of Subsection 6.2 
3. Simulation: this folder contains the code to reproduce the tables and figures of Subsection 5
